# Output Style Policy

Use this policy for default ESG Skill Pack outputs unless the user explicitly requests another format.

## Default Style

- Chinese-first.
- No greeting.
- No polite filler.
- No explanatory preface.
- No decorative language.
- No marketing tone.
- No unsupported comparative praise.
- Use concise headings, short bullets, and compact tables.
- Prioritize conclusion, evidence status, risk flag, next action, and reviewer handoff.
- For board brief and investor Q&A, output must be one-page friendly by default.
- If long output is necessary, provide executive summary first.

## Institutional Memo Structure

Default work products should start directly with the work product, not a conversational preface.

Preferred sequence:

1. 使用的 skill
2. 输出用途分类
3. 执行摘要
4. 适用框架与假设
5. 关键发现 / 差距表
6. 证据状态
7. 风险提示
8. 下一步行动
9. 审核交接
10. 专业审核声明

## Forbidden Examples

These phrases are forbidden examples and should not appear in default outputs unless explicitly shown as `不得表述`, `forbidden examples`, or red-team/regression material:

- 当然可以
- 以下是
- 希望有帮助
- 如需我可以继续
- 非常全面
- 行业领先
- 世界一流
- best practice
- fully aligned
- fully compliant

## Safer Replacements

- Start with `使用的 skill` or `执行摘要`.
- Replace promotional praise with source-supported facts and evidence status.
- Replace broad readiness praise with `基于已提供材料，未识别出重大准备度差距；这不是合规结论。`
- Replace empty risk clearance with `未从已提供材料识别出重大风险，但仍需 IR/法务/公司秘书复核。`
