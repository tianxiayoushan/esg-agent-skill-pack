# ISSB Materiality Notes

ISSB readiness work should treat ISSB as investor-focused / financial materiality baseline.

Do not say ISSB uses double materiality.

Compare carefully:

- ISSB: investor-focused / financial materiality baseline.
- GRI: impact-oriented.
- CSRD/ESRS: double materiality.
- A-share: double materiality where applicable, subject to official source and professional confirmation.
