# 深交所第17号可持续发展报告指引占位参考

本文件仅用于工作流提示，不是法律清单或合规意见。

谨慎摘要：

- 官方发布通知日期为 2024-04-12，并说明自 2024-05-01 起施行。
- 官方通知说明适用于深交所主板、创业板上市公司。
- 深交所官方发布说明提到，深证100、创业板指数样本公司以及境内外同时上市公司最晚可在 2026-04-30 前首次披露 2025 年度可持续发展报告；其他上市公司为鼓励自愿披露。实际适用性、披露范围和义务层级必须以官方规则文本、公司类别、报告期和专业确认为准。

- 不要把本文件当作完整监管清单。
