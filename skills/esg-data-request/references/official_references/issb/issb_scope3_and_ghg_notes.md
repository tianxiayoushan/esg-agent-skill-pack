# ISSB Scope 3 And GHG Notes

Readiness checks should request:

- Scope 1, Scope 2, and Scope 3 activity data and calculation methodology.
- Reporting boundary and consolidation approach.
- Scope 3 category coverage and materiality rationale.
- Data quality, estimation methods, emission factors, and calculation owner.
- Assurance status stated separately from evidence status.
- financed emissions applicability should not be assumed unless financial activities are material and professionally confirmed.

Do not infer emissions figures, categories, assurance, or completeness from partial evidence.
