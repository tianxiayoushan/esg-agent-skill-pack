# Official Rules Reference Pack

This folder provides official-source maps for HKEX, A-share, and ISSB ESG / sustainability disclosure readiness workflows.

It is designed to help agents ask fewer repeated framework questions and keep framework references traceable. It does not turn ESG Skill Pack outputs into legal, regulatory, audit, assurance, financial, sustainability reporting, filing, or compliance opinions.

## Use Rules

- Use official source URLs as the authority pointer.
- Do not copy full official rules or standards into outputs.
- Treat summaries as navigation aids only.
- Confirm issuer category, reporting period, framework adoption, obligation level, and applicability with responsible reviewers before external use.
- Keep outputs as readiness workpapers unless the user provides professional confirmation and approved source text.

## Folder Map

- `hkex/`: Appendix C2 / ESG Reporting Code, Part D climate source maps, and HKEX obligation matrix.
- `a_share/`: SSE No.14, SZSE No.17, subject-scope matrix, topic map, and A-share obligation matrix.
- `issb/`: IFRS S1 / IFRS S2 source maps and ISSB readiness notes.
- `crosswalks/`: HKEX, A-share, ISSB, GRI, and CSRD/ESRS comparison maps.
- `disclaimers/`: copyright/source-use and professional-review policies.
